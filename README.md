# 🎉 Sri Durga Events

A modern, responsive luxury event management business website designed to showcase premium event planning services with an elegant and user-friendly interface.

---

## 🚀 Features

- Luxury Hero Image Slider
- Responsive Design
- Smooth Animations & Transitions
- Featured Services
- Gallery Showcase
- Packages Section
- Contact Section
- WhatsApp, Phone, Email & Instagram Integration

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- Vite
